# StudyBot-LLM
An LLM-powered college student assistant built with Streamlit and the Hugging Face Inference API.
Supports multi-turn conversation with context retention, topic filtering to restrict off-topic queries, and multiple model options (Llama 3.1 8B, Llama 3.3 70B, DeepSeek R1) via the Hugging Face Router. 

Required to create a hugging face token before use (can save locally in dotenv file or input manually as of use)
