# StudyBot-LLM
An LLM-powered college student assistant built with Streamlit and the Hugging Face Inference API.
Supports multi-turn conversation with context retention, topic filtering to restrict off-topic queries, and multiple model options (Llama 3.1 8B, Llama 3.3 70B, DeepSeek R1) via the HF Router. Built as a contrast to a rule-based approach to demonstrate the tradeoffs between deterministic and generative AI systems.
